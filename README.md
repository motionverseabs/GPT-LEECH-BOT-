# Advanced Public Leech Bot

## Deploy on Railway
1. Fork / upload this repo to GitHub
2. Create Railway project → Deploy from GitHub
3. Set ENV variables:
   - BOT_TOKEN
   - API_ID
   - API_HASH
   - OWNER_ID
4. Done ✅

## Commands
/start /help /ping and admin basics included.
